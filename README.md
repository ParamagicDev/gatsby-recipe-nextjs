# Gatsby Recipe Next.js

A silly idea I had to use Gatsby Recipes to install Next into a project.

## Add to a Project

```shell
gatsby recipes https://raw.githubusercontent.com/marcuslyons/gatsby-recipe-nextjs/master/react-leaflet.mdx
```
